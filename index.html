What is the document object Model (DOM) ?

(DOM) is a programming interface for web documents. It represents the structure of a document as a tree of objects, allowing scripts to manipulate the content, structure, and style of web pages dynamically.

How do you select an element from the dom?

To select an element from the DOM, you can use methods like getElementById, querySelector, or getElementsByClassName.
